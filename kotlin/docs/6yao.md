## 吉凶占

吉凶占：特指占人或生物的平安、健康、疾病等方面的吉凶。
这里的吉凶是狭义的，不是广义的“好坏”。

吉凶占的四个步骤：
1. 取用神
2. 看旺衰
3. 看生克
4. 得结果

用神吉凶的八种情况
- 用旺
  - 用旺逢生：锦上添花，吉
  - 用旺受克：易有不利，凶
  - 用旺无克，平平健康，吉
- 用衰
  - 用衰逢生：危中有救，吉
  - 用衰受克：屋漏逢雨，凶
  - 用衰无克：勉力支撑，吉
- 极端
  - 用神旺极：刚脆则折，凶。不常见，回光返照。
  - 用神弱极：衰极无救，凶
